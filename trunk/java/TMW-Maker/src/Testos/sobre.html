TMW-MAKER

VERSÃO JAVA (GLP-v3)

O TMW-Maker é um software tipo Engine, que ajudará no desenvolvimento do jogo MMORPG Open Source The Mana World, facilitando a inclusão e edição códigos de NPCs(Eathena Script), MOBs(monstros), WARPs(Portais), XML(Itens) até por leigos em programação através de menus intuitivos.
